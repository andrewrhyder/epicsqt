void QCaFloating::writeFloating ( const double &  data) [slot]

Take a new floating value and write it to the database. The type of data formatted (text, floating, integer, etc) will be determined by the record data type, How the floating is parsed will be determined by the floating formatting. For example, floating to string may require always including a sign.
