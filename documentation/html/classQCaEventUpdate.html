Detailed Description

Packages CA callback data as a Qt event.

This class packages data and status generated by a CA callback as a Qt event. When data or status updates must be passed from a foreign CA thread to a Qt thread an instance of this class is created, populated with the CA information and posted through the event mechanism to a Qt thread.
