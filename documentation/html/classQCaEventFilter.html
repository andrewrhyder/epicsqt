Detailed Description

CA event filter for QT.

CA Date Time manager.

Manage CA connection information.

Manage CA alarm and severity information.

This class allows instances of QCaObject to recieve events posted to them by foriegn CA threads. Each event recipient creates an instance of this class. This class, however, manages only a single event filter for all instances of itself.
