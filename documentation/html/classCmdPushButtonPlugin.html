QString CmdPushButtonPlugin::variableSubstitutions [read, write]

Variable substitutions Example: SECTOR=01 will result in any occurance of $SECTOR in variable name being replaced with 01. Note, unlike most objects with a variable name substitions, changes to this property do not take immediate effect, so the normal code which waits for the user to finish typing, then propogates the substitutions is not required. the substitutions are simply used next time the button is pushed.
